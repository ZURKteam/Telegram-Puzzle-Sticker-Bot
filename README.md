🧩 Telegram Puzzle Sticker Bot

🎉 Turn your photos & videos into puzzle stickers for Telegram!  
✨ Supports animated video stickers, custom puzzle sizes, and automatic sticker pack creation. Perfect for mosaics, banners, or full sticker sets.

---

⚡ Features

-  📸 **Photo → Puzzle**: Convert any photo into puzzle stickers  
-  🎬 **Video → Animated Puzzle**: Create animated WEBM stickers from videos  
-  ✏️ **Custom Sizes**: Choose your own puzzle layout (e.g., 2x5, 4x10)  
-  🧩 **Auto Sticker Pack**: Bot automatically generates a sticker pack  
-  🗑 **Auto Cleanup**: Temporary files removed automatically  
-  🚀 **Fast & Easy**: Optimized for private/single-user use
